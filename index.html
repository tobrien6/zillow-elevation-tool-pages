What it does

Everything elevation, right inside Zillow.

No new tabs, no copy-pasting coordinates into a separate map. Open Zillow, scroll, see elevation. It's that simple.

Elevation badges on every listing

A small badge appears on each visible Zillow listing card with the approximate elevation in feet or meters.

Matched map pin labels

When the listing is uniquely identifiable on the map, the price pin gets an elevation label too — so you can browse the map view with terrain context.

Pro elevation filters

Set a min and max elevation. Pro auto-hides listings outside your range, keeping pending lookups visible while data resolves.

Local cache for speed

Resolved elevations are cached on your device so revisited listings don't burn through your daily lookup quota.

Multi-source coverage

Integrates with multiple public elevation data sources for excellent coverage across the country, with graceful fallbacks when one is unavailable.

Privacy-respecting

Runs only on Zillow tabs you open. No tracking pixels, no ad networks, no resale of listing data.

How it works

Three steps. No setup beyond installing.

  1. Install the extension

    Add Zillow Elevation Tool from the Chrome Web Store. Pin it to your toolbar so the popup is one click away.

  2. Browse Zillow normally

    Open any Zillow search or listing page. Elevation badges appear on visible cards as you scroll.

  3. Filter with Pro (optional)

    Open the popup, set a min and max elevation, and listings outside your range get hidden automatically.

FAQ

Common questions.

How accurate are the elevation values?

Elevation is sampled from public terrain data (AWS Terrarium tiles with USGS EPQS fallback) at the listing's coordinates. Expect typical accuracy within a few meters in well-mapped areas. Use it for relative comparisons between listings, not for survey-grade decisions.

Does it work on every Zillow page?

It works on Zillow search results, map view, and individual listing detail pages. Listings without coordinates available in the page or map data show a "coordinates unavailable" state — they are not geocoded by default to keep your daily lookup quota meaningful.

What does "lookups per day" mean?

A "lookup" is a new elevation request the extension hasn't cached yet. Cached results don't count, so revisiting the same listing is free. Quotas reset by UTC date.

Is my browsing tracked?

No. The extension only runs on Zillow tabs you open, only sends coordinates to public elevation providers to fetch elevation values, and stores cache and settings locally in Chrome extension storage. See the privacy policy for details.
