Product and Service Description

Atlas Software sells access to Zillow Elevation Tool, a Chrome extension that runs on Zillow pages the user chooses to view. It reads visible listing and map data, estimates elevation from public elevation data sources, and displays elevation badges and matched map labels in the browser.

Free users receive basic elevation display with a daily limit on new elevation lookups. Pro subscribers receive elevation filter controls and a higher daily lookup limit.

Elevation values are estimates and are not survey, engineering, safety, legal, insurance, or flood-risk determinations.
